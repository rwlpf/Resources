
## Migrate SSRS to Power BI Premium

**
Before starting type of migration there are some considerations which should be reviewed. This [page from Microsoft documents](https://docs.microsoft.com/en-us/power-bi/guidance/migrate-ssrs-reports-to-power-bi#migration-tool) is a worth reading.

The first step is to discover what have in current environment. The easiest way to complete this is run 
[Microsoft Assessment and Planning Toolkit](https://www.microsoft.com/en-us/download/details.aspx?id=7826)  

There is a tool which can be used to automatically migrate reports from SSRS -  > https://github.com/microsoft/RdlMigration

Patrick Le Blanc did a video on how to use it (yes its not clear from the documentation) - https://www.youtube.com/watch?v=_VaBzheAgxY
